# Lumoji

Lumoji is a searchable reference for Google's Noto 3D emoji set. Browse 230+ emoji by name or category, see each one's Unicode codepoint and group, and tap any row to copy the emoji to your clipboard. No account, no ads — the full Noto 3D catalog offline.

iPhone & iPad. Free. No ads, no in-app purchases, no accounts, no network.
